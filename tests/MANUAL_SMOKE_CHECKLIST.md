# Manual Smoke Checklist

## Notes tab

- [ ] Open Notes tab successfully
- [ ] Create a note without a linked task
- [ ] Create a note with a linked task
- [ ] Verify linked task metadata appears in note card
- [ ] Search by note title
- [ ] Search by note body
- [ ] Search by tag
- [ ] Click Edit once and verify editor opens immediately
- [ ] Click Save once and verify note updates immediately
- [ ] Click Cancel once and verify editor closes immediately
- [ ] Click Delete once and verify confirm state appears immediately
- [ ] Click Confirm delete once and verify note disappears immediately
- [ ] Click Open Note from Home and verify matching note auto-expands in Notes

## Home tab

- [ ] Recent Activity displays two columns
- [ ] Recent Tasks renders task activity correctly
- [ ] Recent Notes renders note title, preview, timestamp
- [ ] Open Task from Home stores selection correctly
- [ ] Open Note from Home stores selection correctly

## Board tab

- [ ] Open Task from Home and switch to Board
- [ ] Verify the selected task is preselected immediately
- [ ] Verify Board shows "Opened from Home dashboard." when applicable
- [ ] Save task changes successfully
- [ ] Mark task done successfully
- [ ] Add task note successfully
- [ ] Verify task note history updates
- [ ] Verify time logged total updates

## Regression checks

- [ ] No two-click behavior on edit/save/delete
- [ ] No infinite rerun behavior
- [ ] No broken selectbox rendering
- [ ] No missing form submit warnings
- [ ] No import errors on app startup
